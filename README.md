# Financial Report - Power BI

## Sobre o Projeto

Este projeto consiste na reformulação de um relatório financeiro desenvolvido no Power BI, com foco na melhoria da **experiência do usuário (UX)** e na organização visual das informações.

O objetivo foi transformar um relatório básico em um dashboard mais intuitivo, interativo e visualmente agradável.

---

## Objetivos

- Melhorar a usabilidade do relatório
- Aplicar boas práticas de design
- Facilitar a interpretação dos dados
- Tornar a navegação mais intuitiva

---

## Conceitos Aplicados

Durante o desenvolvimento, foram considerados os seguintes princípios:

- Posicionamento estratégico dos elementos
- Contraste para melhor leitura
- Proporção áurea no layout
- Segmentação de dados
- Foco na experiência do usuário (UX)

---

## Funcionalidades do Relatório

- Dashboard com indicadores financeiros
- Gráficos interativos
- Segmentação de dados por filtros
- Navegação entre páginas com botões
- Matriz de detalhamento de vendas

---

## Estrutura do Relatório

O relatório é composto por **3 páginas**, contendo:

1. Visão geral dos dados financeiros
2. Análise detalhada
3. Navegação interativa com botões

---

## Melhorias Realizadas

- Ajuste de cores e contraste
- Organização dos visuais
- Padronização do layout
- Criação de botões de navegação
- Destaque para informações mais relevantes

---

## Como Utilizar

1. Baixe o arquivo `.pbix`
2. Abra no Power BI Desktop
3. Interaja com os filtros e visuais
